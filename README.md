# silent.dolphin

A Google apps script to export mails from gmail to a google doc. Great to feed such a generated document to a LLM such as Notebook LM.

## Description

a simple script, that reads emails from gmail based on labels and then stores them in a google doc.

## usage

- update the config.gs with the labels you want and add the name of the document you want the information stored in
- create a regular task
- and ... done
